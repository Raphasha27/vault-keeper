# Vault Keeper

Local-first encrypted password manager with PBKDF2 key derivation, TOTP support, and auto-lock

## Features

- cli
- cryptography
- encryption
- password-manager
- security
- totp

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Raphasha27/vault-keeper.git
cd vault-keeper
```
